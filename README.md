# Study Time Tracker

This project is a simple Python program that estimates your weekly study hours based on how many hours you studied today. It demonstrates basic user input, numeric conversion, calculations, formatted output, and error handling.  
The project also shows a clean workflow using Git + GitHub with meaningful commits.
## How to Run the Program

Make sure you have Python installed, then run:
app.py
